Advance Metering Infrastructure TMR 2012 AMI System

Is the latest version of Microstar's AMI software, it provides one solution that combines the functionalities of Automated Meter Reading (AMR), Automatic Meter Management, Metering Data Management (MDM) Business Intelligence (BI) and more. It supports 2 way communictaion between head-end system and smart meters in the field. Both remote meter reading and remote meter management are supported.

TMR-2012 provides a web-based end user interface. Users of the system with proper authentications and authorizations can access the system as long as they have network connection and a browser.

  • User Friendly Web-based UI
  • All Integrated in One System
  • Report, Billing and Business Intelligence
  • Electricity Vending System Integrations
  • Customer Portal
  • Scalable and Cross Platform
  • Open System & Multi-Vendor Compliant
  • Data Query
  • System Integration
